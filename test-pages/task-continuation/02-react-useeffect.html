What is useEffect?

The useEffect hook is used for side effects in React functional components. It's similar to componentDidMount, componentDidUpdate, and componentWillUnmount combined.


useEffect(() => {
  // Your side effect here
  console.log('Effect runs!');
  
  return () => {
    // Cleanup function
  };
}, [dependencies]);

Dependency Array

The second argument to useEffect controls when your effect runs:

  • [] - Runs once on mount (like componentDidMount)
  • [dep1, dep2] - Runs when dependencies change
  • No array - Runs after every render

Common useEffect Patterns in React

Use useEffect for:

  1. Fetching data from APIs in React
  2. Setting up subscriptions in React components
  3. Manually changing the DOM in React
  4. Setting up timers in React

The useEffect hook integrates perfectly with useState and other React hooks.

Cleanup Functions

Always clean up subscriptions, timers, and event listeners in React useEffect to prevent memory leaks in your React application.


useEffect(() => {
  const subscription = api.subscribe();
  
  return () => subscription.unsubscribe();
}, []);
